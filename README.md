# Recipe Journal

A simple website to save and organize personal recipes. Users can add recipes with ingredients and instructions and keep them in one place.

## Features

* Add new recipes
* View saved recipes
* Simple clean interface
* Easy to expand with more features later

## Technologies Used

* HTML
* CSS
* JavaScript

## Project Structure

```
recipe-journal
│
├── index.html      # main webpage
├── style.css       # website styling
├── script.js       # website logic
├── recipes.json    # recipe data storage
└── README.md       # project documentation
```

## How to Run the Project

1. Download or clone the repository from GitHub.

2. Open the project folder.

3. Double-click the file:

```
index.html
```

4. The website will open in your browser.

## How to Work on the Project

1. Open the project folder in a code editor such as Visual Studio Code.

2. Edit files depending on what you want to change:

* `index.html` → add or change page structure
* `style.css` → change colors, layout, or fonts
* `script.js` → add new features like saving or searching recipes

3. Save the file.

4. Refresh the browser to see your changes.

## Future Improvements

* Add search for recipes
* Add categories (dessert, dinner, etc.)
* Add images for recipes
* Save recipes to a database
